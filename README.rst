Mopidy-Radio
=============

`Mopidy <http://www.mopidy.com/>`_ extension for playing radio from
`radio.de <http://www.radio.de/>`_.


Usage
-----

#. Install the Mopidy-Radio extension by running::

    sudo pip install mopidy-radio
